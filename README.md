# RealChat-Agent (真实的男大学生 AI 智能体)

这是一个基于真实微信聊天数据集（近7000条记录）逆向工程打造的 **高保真数字孪生 / 垂直领域 AI Agent**。
本项目通过提取真实人类的词频、回复长度分布、口头禅以及专业领域知识（CSGO饰品市场、电竞比赛、校园生活等），结合 Gemini 3.5 Flash 引擎和微信底层协议，实现了一个**完全没有AI痕迹**的聊天机器人。

## 🌟 核心特性

- **🤖 零AI痕迹 (Zero AI Signature)**
  - 严格通过 Prompt Engineering 剥离了大模型的“礼貌感”与“爹味”。
  - 禁用长篇大论、书面语和常规标点符号，完全符合当代大学生的微信聊天习惯。
  - **精准的话痨/高冷比例**：基于真实数据统计（24%超短句，43%短句，绝不长篇大论）。

- **🧠 深度垂直领域知识 (Domain-Specific Knowledge)**
  - 内置了极其专业的 CSGO 饰品交易逻辑（大盘走势、抄底、割肉、红线AK、BUFF交易技巧）。
  - 拥有对具体游戏（CS2实战、开箱概率）、电竞（Faze、绿龙战队）和日常数码外设的深刻理解。
  - 甚至包含了“穷但想加仓就用花呗”的真实人类赌徒心理模型。

- **⚡ 毫秒级极速响应 (Ultra-Low Latency)**
  - 核心引擎采用 `gemini-3.5-flash`，结合极致压缩的上下文 (History Trim) 和 `max_output_tokens` 限制，消除思考延迟。
  - **拟真打字节奏 (Jitter)**：多段消息会自动拆分（通过 `|||` 魔法符）并带有人性化的随机延迟发送机制，模拟真人连续打字。

- **🖼️ 全媒体响应支持 (Multi-Media Handling)**
  - 彻底解决了普通文本机器人的“睁眼瞎”问题。
  - 能够智能接收并针对用户的**表情包、图片、视频**给出符合人设的自然反馈（如“这是啥”、“牛逼”或反手回敬一个专属表情包）。

## 🚀 快速启动

### 环境要求
- Windows 系统
- Python 3.9+ (安装时请务必勾选 `Add Python to PATH`)

### 安装与运行

1. 下载或 Clone 本仓库。
2. 打开 `config.py`，配置你的 API 参数：
   ```python
   # 申请地址: https://aistudio.google.com/app/apikey
   GEMINI_API_KEY = "YOUR_API_KEY_HERE"
   
   # 修改为你想要机器人回复的微信好友备注或昵称
   ALLOWED_FRIEND_NAMES = [
       "好友A",
       "文件传输助手",
   ]
   ```
3. 直接双击运行 `运行机器人.bat`。
   *(脚本会自动检查环境、配置清华源并安装所需要的 `itchat-uos` 和 `google-genai` 依赖)*
4. 在弹出的图片查看器中**使用微信小号扫码登录**。

## 🛠️ 技术架构

- **底层通信**：[itchat-uos](https://github.com/why2lyj/ItChat-UOS) 网页版微信协议
- **AI 引擎**：Google Gemini `gemini-3.5-flash` (google-genai SDK)
- **数据处理**：Pandas, Jieba (用于前期对 30万字 历史聊天记录的实体抽取和词频统计)

## 📝 自定义你自己的数字孪生

如果你想把这个机器人改造成你自己的性格，请修改 `persona.py` 文件：
1. **替换【你们的共同世界】**：将里面的 CSGO 知识替换为你和朋友常聊的领域（比如二次元、打篮球、美妆）。
2. **替换【高频回复原文】**：把你平时最爱说的 10 个词写进去。
3. **替换【表情习惯】**：把你最爱发的微信表情包文本（如 `[捂脸]`、`[吃瓜]`）加进去。
